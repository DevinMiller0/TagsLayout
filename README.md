# TagsLayout（FlowLayout）
## Android自定义ViewGroup 标签流仿天猫

# 支持：
* **支持setOnTagsOnClickListener，回调标签上显示的text**
* **自定义背景，字体大小颜色，设置tag之间的距离（margin），tag的padding**
